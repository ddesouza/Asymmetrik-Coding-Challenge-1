This program is made for the Coding Challenge question 1.

In order to run this program, import the "ChallengeSolutionZip" Java Project into your IDE of choice (I used Eclipse to write this), and run the "Driver.java" file within the project.

INSTRUCTIONS FOR USING THE PROGRAM
Input business card information, line-by-line. Press ENTER to input a new line of information. When finished, type DONE on a single line and press ENTER to see results.
